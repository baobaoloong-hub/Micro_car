#### 说明
1. 编码器测试过是可以的，`编码器磁铁和电机靠的有些近会有一些的小影响，自制编码器没什么难点，一个磁编码器芯片，需要输出方向正交编码器加个 d 触发器就可以了，可以试试英飞凌的 tle5012，e1000 版本的，出厂默认正交输出（手册好像写的 4096 线），也可以使用其他的比如 mt6835...

2. 电机选型(预留了10mm,12mm孔位,其他的自行修改)，没细找电机，直接无脑 030 ,最好是铷磁铁的，扭力要大（折线需要大扭力，尤其是直角，有时候跑不快，上限低大概率是电机没劲）,牛爷爷用的那个也挺好，就是略微有点重。有实力的佬挑战一下无刷，好盈 1625，一些蚊车电机，淘宝还有好多不知名的好电机，钞能力直接马克松冯哈勃，二手的不推荐,好多都是快坏了的

3. 裙边，材质自己挑，铁氟龙的看过学弟的，感觉偏硬，容易变形，没用过不评价,用过的可以说说，推荐那种大棚膜，2 平米 2.6 元，四舍五入就是白送！根本用不完，耐磨还有韧性不易形变，强推强推，还有就是贴膜，直接纳米胶，大块的铺，尽量铺宽一点，不要留缝防止漏气，可以参考我给的图片。

4. 负压...

5. 开启负压后车模变形，轮子出现外八问题，不一定要觉得是底板硬度不够，可以再上面加个支撑，这个支撑可以是你的 板子，也可以任何东西，参考我去年的车模结构，底板我有打过铝基板，单层正好能放开电源和三个电机的驱动，1.5mm 板很重，20g 整，有佬有空可以试试 1mm 或者换成普通的板子。还有，飞檐走壁不知道底盘是不是有点低，自行调整（我感觉没大问题）

6. 胎皮，省事直接买的成品，很好用，推荐一下牛爷爷的硅胶贴包海绵（牛爷爷群里有）和青山佬的世一胎皮，

7. 3d 打印部分，我用的竹子的 a1mini，打印部分只有扇叶和负压支架，一定要开圆孔补偿，可以轻松装上且扇叶不松动，想要减轻重量轮毂也可以打印。
关于负压电机的那个支架，可以改进一下，换成弹簧型那种，我看到过国外的鼠车并不是直接连接，好像因为扇叶动平衡不好，会震动，螺丝共振会松动，陀螺仪可能也会漂，有条件的陀螺仪也加一下减震柱，类似穿越机（可能有点多此一举），不过这些问题我还没有发现车模也没感觉到震动

8. 关于车模的测试，我是建议大家一步到位，能用碳板用碳板，不要用 3 d 打印，因为很多问题是当你换了稳定的材质才发现根本没那回事，尤其是负压和底板摩擦，一套碳板大概 40 左右甚至更低，不想花钱直接打 pcb 替代碳板（比赛哪有不花钱的）
